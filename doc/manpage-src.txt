NAME
  bindechexascii -  converts ASCII, binary, octal, decimal, hexadecimal and prints results on the screen
SYNOPSIS
  bindechexascii [options] arg1 arg2 arg3
DESCRIPTION
  bindechexascii is a program that converts the argument from standard input to the numbering system or character set (ASCII) specified in the option and prints the result on the screen. That way you can perform the conversion between ASCII, binary, octal, decimal, hexadecimal.

OPTIONS

   -h, --help        Print this help
   -i, --info        Print information about program

   --b2o             Binary to octal
   --b2d             Binary to decimal
   --b2h             Binary to hexadecimal
   --b2a             Binary to ASCII

   --o2b             Octal to binary
   --o2d             Octal to decimal
   --o2h             Octal to hexadecimal
   --o2a             Octal to ASCII
 
   --d2b             Decimal to binary
   --d2o             Decimal to octal
   --d2h             Decimal to hexadecimal
   --d2a             Decimal to ASCII
 
   --h2b             Hexadecimal to binary
   --h2o             Hexadecimal to octal
   --h2d             Hexadecimal to decimal
   --h2a             Hexadecimal to ASCII
 
   --a2b             ASCII to binary
   --a2o             ASCII to octal
   --a2d             ASCII to decimal
   --a2h             ASCII to hexadecimal
 
EXAMPLES

  To convert the number 1000111 from binary to decimal:

       bindechexascii --b2d 1000111

  To convert the number 131 from octal to binary:

       bindechexascii --o2b 131
 
  To convert the number 50 from decimal to binary:

       bindechexascii --d2b 50
 
  To convert the number 3F from hexadecimal to binary:

       bindechexascii --h2b 3F
 
  To convert the caractere 'a' from ASCII to binary:

       bindechexascii --a2b a
